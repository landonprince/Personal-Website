Portfolio

  • Pixalyze

    C++ | CMake | OpenCV

    Pixalyze is an easy-to-use command-line application. It allows for the use of an ever-growing collection of commands to analyze and process images. This application primarily utilizes the OpenCV library to rapidly parse and manipulate images in real-time.
    Github

  • PySlice

    Python | Pygame

    Pyslice is an addictively fun 2D platformer made entirely with Python. Play as a katana-wielding ninja to slice through your enemies and traverse dangerous worlds. It utilizes the Pygame library to provide a robust and seamless experience for the player.
    Github

  • Personal Website

    HTML | CSS | JavaScript

    My personal website (this website) displays my portfolio, information about me, and my contact information in a visually appealing manor. Tailwind CSS and NodeJS were used in addition to vanilla HTML/CSS/JS to construct an elegant, responsive website.
    Github

  • Mad Libs Generator

    Java | JUnit

    Mad Libs Generator is a program that allows the player to generate their own custom Mad Lib templates, and then fill in the blank spaces to create fun stories. Players can choose the number of sentences, frequency of blank spaces, and the overall theme of the Mad Lib.
    Github

  • ChatFlow Manager

    Java | JUnit

    ChatFlow manager is a program that allows the user to store, retrieve, and manipulate messages. There are two core components to this program: the Message object that contains data and the database object that holds a linked list of Message objects.
    Github

  • Image To ASCII

    C++ | CMake | OpenCV

    Image to ASCII is a program that uses the OpenCV library to allow users to convert nearly any image to high-quality ASCII art. The ASCII art is generated and returned in a text file. Simply enter the file path of the image you want to convert and enjoy the ASCII art.
    Github
